# MTG Collection Manager

Progetto didattico per il corso di Piattaforme Software per Applicazioni su Web.

## Descrizione

MTG Collection Manager è una piattaforma web per la gestione di collezioni e mazzi di Magic: The Gathering.

L'applicazione consente agli utenti di cercare carte tramite un servizio REST esterno, aggiungerle alla propria collezione, creare mazzi personali e scambiare messaggi persistenti con altri utenti.

## Tecnologie previste

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Hibernate
- PostgreSQL
- Spring Security
- Keycloak / JWT
- Angular
- TypeScript
- REST API

## Struttura del progetto

- `backend/`: applicazione Spring Boot
- `frontend/`: applicazione Angular
- `docs/`: documentazione del progetto
